# Handy для русской речи — установка и настройка

Это форк [cjpais/Handy](https://github.com/cjpais/Handy) с адаптацией под русскую речь через Groq Whisper (`whisper-large-v3`) — качество распознавания заметно выше, чем у локальных моделей.

Оригинальный Handy — open-source приложение для голосового ввода на Mac/Windows/Linux. Ты нажимаешь горячую клавишу, говоришь, отпускаешь — распознанный текст вставляется в активное поле любого приложения.

## Что добавлено в этом форке

- **Groq Whisper** как основной движок распознавания русского (модель на серверах Groq, локально ничего не крутится)
- **WAV-чанкинг 25 секунд** — длинные записи режутся на куски, каждый транскрибируется отдельно; это убирает обрезание конца длинных сообщений
- **VAD-обрезка тишины в конце** — Silero VAD находит последний фрейм речи и обрезает всё что после
- **Слияние огрызка** — если после чанкинга остаётся куск < 5 сек, склеивается с предыдущим (иначе Whisper галлюцинирует "Продолжение следует…")
- **Фильтры галлюцинаций** — ловят типичные YouTube-эндинги ("Субтитры создавал…", "Редактор субтитров…", trailing "Да."/"Нет." после вопроса)
- **Автоматическое разбиение на абзацы** для длинных транскрипций

Подробности патчей — в [OUR-PATCHES.md](OUR-PATCHES.md).

## Установка

### 1. Скачай сборку под свою систему

На странице [Releases](https://github.com/leonovs0808-star/Handy/releases) возьми файл под свою платформу:

- **Mac Apple Silicon** (M1/M2/M3/M4): `Handy_x.x.x_aarch64.dmg`
- **Mac Intel**: `Handy_x.x.x_x64.dmg`
- **Windows 10/11**: `Handy_x.x.x_x64-setup.exe`
- **Linux**: `.AppImage` или `.deb`

### 2. Установи

**Mac:**
1. Открой скачанный `.dmg`
2. Перетащи Handy в Applications
3. При первом запуске macOS скажет "неопознанный разработчик" — это нормально, приложение подписано ad-hoc
4. Правая кнопка мыши по Handy → **Open** → **Open Anyway**
5. Разреши доступ к микрофону и Accessibility, когда попросит

**Windows:**
1. Запусти скачанный `.exe`
2. SmartScreen скажет "Windows защитил ваш ПК" — это тоже нормально
3. Жми **More info** → **Run anyway**
4. Пройди установщик

### 3. Получи Groq API-ключ

Ключ бесплатный, есть щедрый бесплатный лимит.

1. Зайди на [console.groq.com](https://console.groq.com)
2. Зарегистрируйся (Google/email)
3. **API Keys** → **Create API Key** → скопируй строку `gsk_...`

### 4. Вставь ключ в Handy

1. Открой Handy
2. **Settings** → раздел **Post-Processing** или **API keys** (в зависимости от версии)
3. Поле **Groq API key** — вставь скопированный `gsk_...`
4. В **Models** выбери **Groq Whisper Large v3**
5. В **Language** выбери **Russian**

Готово.

## Как пользоваться

1. Кликни в любое текстовое поле (браузер, редактор, чат, что угодно)
2. Зажми `Option + Space` (Mac) или `Ctrl + Space` (Windows/Linux)
3. Говори
4. Отпусти — текст вставится сам

Есть альтернативный режим — двойное нажатие: одно включает запись, второе останавливает. Настраивается в Settings → General.

## Частые вопросы

**Groq стоит денег?**
Бесплатный лимит покрывает обычное использование одного человека. Если упрёшься — оплата минимальная, посмотри тарифы на console.groq.com.

**Аудио уходит в Groq — это безопасно?**
Groq — американская компания, аудио уходит на их серверы для распознавания. Если критична приватность — используй встроенные в Handy локальные модели (Whisper, Parakeet), только у них качество для русского хуже.

**Handy молчит / выдаёт ошибку "Groq API error"**
Проверь: 1) API-ключ вставлен и корректный, 2) есть интернет, 3) выбрана модель Groq, а не локальная.

**Обновления**
Когда выйдет новая версия — скачай новый файл со страницы Releases, установи поверх. Настройки сохранятся.
