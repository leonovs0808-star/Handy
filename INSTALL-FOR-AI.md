# Инструкция для ИИ-агента: установить Handy (русская сборка)

Ты помогаешь пользователю установить и настроить **Handy** — приложение для голосового ввода на Mac/Windows/Linux. Это форк с адаптацией под русскую речь через Groq Whisper API. Работай по этой инструкции пошагово, не изобретай альтернативы.

## Твоя роль

Провести пользователя через 6 шагов:
1. Определить платформу.
2. Скачать установщик из GitHub Releases.
3. Установить приложение.
4. Помочь получить бесплатный Groq API-ключ.
5. Настроить Handy.
6. Сделать первую голосовую запись.

## Шаг 1. Определи платформу

Спроси пользователя, если не понятно:
- **Mac или Windows или Linux?**
- Если Mac — **Apple Silicon (M1/M2/M3/M4) или Intel?**

Как узнать чип Mac пользователя: меню Apple → About This Mac → строка "Chip" или "Processor". "Apple M..." = Apple Silicon. "Intel Core..." = Intel.

## Шаг 2. Скачай установщик

Страница релизов: **https://github.com/leonovs0808-star/Handy/releases/latest**

В разделе **Assets** найди файл под платформу:

| Платформа | Имя файла (шаблон) |
|---|---|
| Mac Apple Silicon | `Handy_<версия>_aarch64.dmg` |
| Mac Intel | `Handy_<версия>_x64.dmg` |
| Windows 10/11 | `Handy_<версия>_x64-setup.exe` |
| Linux (универсальный) | `Handy_<версия>_amd64.AppImage` |
| Linux Debian/Ubuntu | `Handy_<версия>_amd64.deb` |

Если можешь запускать команды в терминале пользователя — получи прямую ссылку через GitHub API:

```bash
curl -s https://api.github.com/repos/leonovs0808-star/Handy/releases/latest \
  | grep browser_download_url
```

Иначе — попроси пользователя открыть страницу релизов, найти нужный файл и нажать на него.

## Шаг 3. Установи

### macOS

1. Двойной клик по `.dmg`.
2. Перетащи иконку **Handy** в **Applications**.
3. Извлеки диск: правая кнопка по нему в Finder → **Eject**.
4. Запусти Handy из Applications.
5. **Ожидаемая проблема:** macOS покажет "Приложение не может быть открыто, так как разработчик не может быть проверен". Приложение подписано ad-hoc — это нормально.
   - **Решение:** правая кнопка мыши по Handy в Applications → **Open** → в следующем диалоге снова **Open**. Одноразово.
6. Дай разрешения когда попросит:
   - **Microphone** (обязательно)
   - **Accessibility** (обязательно — для вставки распознанного текста)
   - System Settings → Privacy & Security → соответствующие пункты → включить Handy.

### Windows

1. Запусти `.exe`.
2. **Ожидаемая проблема:** SmartScreen: "Windows защитил ваш компьютер".
   - **Решение:** жми **More info** → **Run anyway**.
3. Пройди мастер установки до конца (обычно всё Next → Install).

### Linux

**AppImage** (универсальный):
```bash
chmod +x Handy_*_amd64.AppImage
./Handy_*_amd64.AppImage
```

**deb-пакет** (Debian/Ubuntu):
```bash
sudo dpkg -i Handy_*_amd64.deb
sudo apt-get install -f  # если сообщит о зависимостях
```

## Шаг 4. Получи Groq API-ключ

Handy использует Groq Whisper (whisper-large-v3) для распознавания русской речи. У Groq есть бесплатный тариф, покрывающий обычное личное использование.

Проведи пользователя по шагам:

1. Открой https://console.groq.com в браузере.
2. **Sign In** → войти через Google или Email.
3. В левой панели навигации выбери **API Keys**.
4. Кнопка **Create API Key** → введи название (например `Handy`) → **Submit**.
5. **Скопируй строку вида `gsk_...` немедленно.** Она показывается один раз, после закрытия окна её не восстановить.
6. Сохрани ключ где-нибудь у себя (в менеджере паролей или в заметке).

## Шаг 5. Настрой Handy

Открой Handy. Пройди по разделам настроек:

1. Открой **Settings** (шестерёнка).
2. **Ввод API-ключа Groq** — не в отдельном разделе "API Keys". Ключ живёт в подразделе Post Process, который скрыт по умолчанию. Точный путь (боковая панель Handy):
   - Зайди в раздел **Advanced** (в боковой панели).
   - Включи тумблер **Experimental Features**.
   - Ниже появится группа **Experimental** с тумблером **Post Processing** → включи.
   - В **боковой панели слева** появится новый пункт **Post Process** → зайди туда.
   - В поле **Provider** выбери **Groq**.
   - В поле **API Key** вставь `gsk_...`.
   - После ввода можно вернуться в Advanced и **выключить тумблер Post Processing** обратно — раздел Post Process уйдёт из sidebar, но ключ сохранится. Наш движок Groq Whisper использует его напрямую независимо от тумблера пост-процессинга.
3. Раздел **Models** → кликни на **Groq Whisper**.
   - **ВАЖНО:** предупреди пользователя, чтобы он **НЕ нажимал "Download"/"Скачать"** у Groq Whisper. Это облачная модель, скачивать нечего. Нужно просто кликнуть на неё для выбора. Если пользователь нажмёт "Скачать" — он увидит `No download URL for model` — это нормально, не баг, просто эта кнопка для облачной модели не работает.
4. Раздел **General** → **Language** → **Russian**.
5. Раздел **Shortcuts** → проверь горячую клавишу. По умолчанию:
   - Mac: **Option + Space**
   - Windows/Linux: **Ctrl + Space**

Изменить можно там же.

## Шаг 6. Первая проверка

1. Открой любое приложение с текстовым полем: браузер, редактор, чат.
2. Кликни в поле ввода.
3. Зажми горячую клавишу.
4. Скажи короткую фразу по-русски.
5. Отпусти клавишу.
6. Через 1-3 секунды распознанный текст должен появиться в поле.

### Если не работает

Пройдись по чек-листу:

- **Ошибка "Groq API error"** — проверь: ключ вставлен полностью (начинается с `gsk_`), интернет работает.
- **Тишина / ничего не происходит** — проверь микрофон в системных настройках, микрофонные разрешения для Handy.
- **Текст не вставляется в приложение** (Mac) — проверь Accessibility permission для Handy: System Settings → Privacy & Security → Accessibility → добавить Handy.
- **Handy распознаёт по-английски** — вернись в Settings → Language → Russian.
- **Смешанный русский+английский, странные слова** — обычная особенность Whisper на смешанной речи, помогает говорить чётче.
